initWithCaller:withParentObject:withSetter:

Set up this unmarshaller at the bottom of a call chain. When parsing nested tags, control is handed to another delegate. When that delegate is done, it will

  • set the parser's delegate to the caller, returning control to it.
  • assign the object it created to the parent field with [parent setter:object].
