Running the S3Uploader Sample

This sample is a browser application that demonstrates how to make requests to Amazon S3 using the SDK.

For a more detailed description of the code, please visit this writeup.

  1. Open the AWSiOSSDK/samples/S3Uploader/S3Uploader.xcodeproj project file in Xcode.
  2. Configure the sample with your AWS security credentials:
    1. Open the Constants.h file.
    2. Modify the ACCESS_KEY and SECRET_KEY definitions with your AWS Credentials.
  3. Run the project.
