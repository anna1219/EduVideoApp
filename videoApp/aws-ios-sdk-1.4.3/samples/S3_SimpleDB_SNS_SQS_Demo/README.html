Running the AWSiOSDemo Sample

This sample is a browser application that demonstrates how to make requests to AWS using the SDK.

  1. Open the AWSiOSSDK/samples/AWSiOSDemo/AWSiOSDemo.xcodeproj project file in Xcode.
  2. Configure the sample with your AWS security credentials:
    1. Open the Constants.h file.
    2. Modify the ACCESS_KEY and SECRET_KEY definitions with your AWS Credentials.
  3. Run the project.
